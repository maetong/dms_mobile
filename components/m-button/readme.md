## m-button  按钮


### 参数
- type{String|primary:蓝色  default:白色  warn:红色  }
- loading{Boolean | 名称前是否带 loading 图标}
- disabled{Boolean | 是否禁用}
- plain{Boolean | 是否镂空}
- size{String | default:默认大小  mini:小尺寸  }
- block{Boolean | 是否有圆角  }

### 代码

```vue
</template>
	<m-button type="primary">页面主操作 Normal</m-button>
</template>
```
